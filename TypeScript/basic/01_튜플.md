## 튜플 (Tuple)

- **정의**: 길이와 각 요소 타입이 고정된 배열
- **용도**: 좌표나 고정 개수의 값 표현

```ts
// 튜플 예시: x, y 좌표
const current: [number, number] = [0, 0];
const target: [number, number] = [4, 5];

// 이동 거리 계산
const dx = target[0] - current[0];
const dy = target[1] - current[1];

console.log(`오른쪽으로 ${dx} 위쪽으로 ${dy} 만큼 이동!`);
// 결과: 오른쪽으로 4 위쪽으로 5 만큼 이동!
```
