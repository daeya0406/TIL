## 튜플 (Tuple)

- **정의**: 길이와 각 요소 타입이 고정된 배열
- **용도**: 좌표나 고정 개수의 값 표현

```ts
let monsterName: string;
let level: number;
let hasGold: boolean;
let skill: undefined;
let area: null;

monsterName = "코드잇 고블린";
level = 255;
hasGold = false;
skill = undefined;
area = null;

// 배열 튜플
const current: [number, number] = [0, 0]; // 갯수 2개로 한정
const current: [number, string] = [0, '문자']; // 배열 각 타입 한정

const items: string[] = [];
items.push("문자");
items.push("추가 문자");
```
